## 2.xml中处理大于号小于号的方法

xml|符号|含义
:--:|:--:|:--:
`&lt;`|<|小于号
`&gt;`|>|大于号
`&amp;`|&|和
`&apos;`|'|单引号
`&quot;`|"|双引号
`&nbsp;`|html|空格

## 4.把已经查到的数据更新
```sql
update a set a.create_time=b.time from test_time a
inner join  (SELECT dateadd(mi,+20,create_time) as time from test_time where create_time >='2021-04-14 15:36:48.000') b 
on a.name=b.name
```
